pysh.transforms.beta.lazyapply module¶

Converts expressions like:

>>> func <<= cat | head > null
    func << (cat | head > null)

The whole expression at the right of the operator is grouped. This allows for callables to receive an argument without using parenthesis.

Warning

This transformation breaks the standard ``<<=` operator in normal Python code. There is no assignament to the operand on the left.
