exception pysh.ExitError(status, message=None)[source]¶

Bases: Exception

Represents an exit status error after running a command.

There are a number of predefined ones named ExitErrorN, where N is a number matching the desired exit status. Additionally ExitN are ready to raise instances of those errors.
