pysh.transforms.alpha.lazyexpr module¶

Infers the requirement of a lambda by detecting which expressions use the ? placeholder.

Only expressions in an assignment or in parameter positions are transformed.

>>> pipe.map( 'foo' + ?.lower() )
  # pipe.map( lambda __1, *args: 'foo' + __1.lower() )
